# 📝 Smart To-Do

A modern and feature-rich To-Do Web Application built using **HTML, CSS, and JavaScript**.
This app helps users efficiently manage daily tasks with a clean UI and powerful features.

## ✨ Features

* ✅ Add, edit, and delete tasks
* 📅 Set date & time for tasks
* ✔️ Mark tasks as completed
* 🔍 Search/filter tasks instantly
* 📊 Progress bar to track completion
* 🌙 Dark / Light mode toggle
* 🔔 Reminder alerts for tasks
* 💾 Data stored using Local Storage
* 📱 Responsive and modern UI

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Modern UI + Glassmorphism)
* JavaScript (DOM Manipulation + Local Storage)
